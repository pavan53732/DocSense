# 🔍 DocSense

> **AI-Powered Documentation Analyzer & Intelligent Co-Architect**

DocSense is a professional-grade **Electron Desktop Application** that transforms unstructured technical documentation into actionable, validated build plans. By combining advanced LLM extraction with a massive 914-rule engineering engine, DocSense ensures your system designs are robust, secure, and implementation-ready.

---

## 🚀 Key Features

*   **🤖 AI-Driven Extraction**: Automatically derives structured system models, user flows, and UI components from raw Markdown/text.
*   **⚖️ 914-Rule Engineering Engine**: Validates designs against 92 categories including Security, Accessibility (A11y), Performance, and SEO.
*   **🛠️ 507 Auto-Fixes**: Automatically corrects over 55% of detected issues with safe, context-aware suggestions.
*   **🖥️ Desktop-First Security**: Specialized hardening for Electron/Windows apps, including IPC security and file system safety.
*   **🧬 State Machine Derivation**: Automatically identifies missing states (loading, error, offline) in user flow specifications.
*   **🎯 Deep Analysis Missions**: Run targeted agents for Security, Performance, and Requirement Traceability.
*   **💥 Change Impact Analysis**: Visualize the "blast radius" of specification changes across your entire architecture.

---

## 🏗️ Technical Stack

*   **Runtime**: Electron 28.x (Node.js + Chromium)
*   **Frontend**: Next.js 16 (App Router), React 19, TypeScript
*   **Styling**: Tailwind CSS 4, shadcn/ui, Lucide Icons
*   **State Management**: Zustand
*   **AI Layer**: OpenAI-compatible API integration with parallel chunked processing
